Course Curriculum

A comprehensive journey through Python programming, algorithms, and AI

01

Introduction

Learn the basic concepts of programming. Introduction to basic program structure, data types, and arithmetic and logical operations. Get familiar with the online judge system.

Practical Projects:

  • Write a basic program with input/output from the screen
  • Perform basic arithmetic operations
02

Conditional Statements

Introduction to conditional structures and branching: concepts and key considerations.

Real-world Problems:

  • Check if a year is a leap year
  • Calculate monthly electricity bills based on usage
03

Loops

Introduction to loop structures: concepts, types, and loop processing techniques.

Real-world Problems:

  • Calculate tiles needed to floor a room
  • Calculate the capacity of a train
04

Functions

Learn about subroutines: concepts, structure, and purpose. Introduction to local and global variables, and how to organize program files.

Common Functions:

  • Check if a number is prime
  • Find the greatest common divisor (GCD) of two integers
05

Arrays

Learn about arrays and one-dimensional arrays: concepts, element access, and processing techniques. Introduction to error checking methods and fast problem-solving approaches.

Real-world Problems:

  • Solve the fruit shopping problem in a supermarket
  • Manage employee ID codes in a company
  • Find which post has the most likes
06

Matrix - 2D Arrays

Learn about two-dimensional and multi-dimensional arrays: concepts, element access, and processing techniques.

Real-world Problems:

  • Play chess
  • Count likes on a photo album on social media
07

String

Learn about strings and character types: concepts and programming techniques with strings. Introduction to ASCII table.

Practical Exercises:

  • Standardize names
  • Count distinct characters
08

Recursion

Learn about basic recursion techniques: concepts, types, and recursive techniques. Basic introduction to the Stack data structure.

Classic Problems:

  • Find Fibonacci numbers
  • Convert between binary and decimal
09

Data Abstraction

Learn about struct/class: Apply struct/class to create new data types to avoid confusion and manage programs easily. Introduction to getters and setters for accessing and manipulating attributes.

Real-world Problems:

  • Search and store student/employee information
  • Find the farthest point in a 2D coordinate system
10

Sorting Algorithms

Learn about classic sorting algorithms. Brief introduction to algorithm complexity.

Real-world Problems:

  • Sort student IDs and scores
  • Help Vasya build a tower
11

Singly Linked List

Introduction to linked lists, FIFO and LIFO singly linked list storage, basic operations on singly linked lists. Pointers: Additional lesson for C++ language.

Problems:

  • Find available hotel rooms based on given information
  • Find the reversed linked list
12

Trees

Introduction to hierarchical Tree data structure in programming: definition, types, and basic concepts. Learn about Binary Tree structure, especially Binary Search Tree: operations and traversal methods.

Problems:

  • Find the student with the highest score
  • Visualize binary tree from top view
  • Level order traversal of binary tree
